# Démarrer un nouveau projet

Démarrer un nouveau projet est très rapide avec Jopi Rewrite, grâce à un outil nommé `jopi`permettant d'installer un template parmi une liste de projets types.

**Commande shell pour sélectionner et installer un projet type**
```
npx jopi init
```

**Equivalent avec bun.js**
```
bunx jopi init
```

> Prérequis: ici il vous faudra uniquement une version récent de node.js (v24.11 minimum) et optionnellement une version récente de bun.js (v1.3 minimum).
